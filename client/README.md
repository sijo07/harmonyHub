# Cheers! - Music App

A modern, immersive web music player built with React, Vite, and Tailwind CSS.

## Features
- **Immersive Design**: Dark "Red and Black" aesthetic with glassmorphism effects.
- **Full Screen Player**: Visualizers, lyrics view, and large album art.
- **Queue Management**: Manage your listening queue with a slide-out sidebar.
- **Responsive**: Fully responsive sidebar and layout.

## Technologies
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
